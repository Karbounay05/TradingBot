# Trader Bot

A simple trading bot implemented in C++ with embedded Python strategy support via `pybind11`.

---

## Overview

This project demonstrates a trading bot architecture with:

- C++ core components for data loading, strategy bridging, and trade execution.
- Python strategy logic embedded and called from C++ using `pybind11`.
- CSV market data loading.
- Executing simple BUY/SELL/HOLD actions based on Python strategy.
